具体层数参考 (针对 8B 模型)：建议重点关注 第 20 层到第 30 层 之间。
快速筛选法：使用与前一层残差流的差分指标，即当两层之间的残差流差异最大时，说明出现了强因果特征 
策略：如果单层干预效果不佳，可以尝试对 Top-3 优质层 进行联合干预，以对抗“九头蛇效应”（模型自我修复）


介入的 Token 时机 (Token Timing)
干预的时机决定了你是在修改“理解”还是修改“决策”。
关键实机：最后一个 Token (Last Token position)。理由：在 Prompt 结束处或逻辑节点处，残差流融合了之前所有的上下文信息，是决策（生成下一个 Token）的爆发点。 
针对逻辑推理任务 (C)：应在 逻辑触发词（如 "So", "Therefore", "Step 1"）出现的 Token 位置进行采样和干预。
在代码任务中，则是在 赋值符号（=）或 分支控制词（if, return）之后。

干预得分的量纲 (Steering Score/Ranking)
激活强度分 (Activation Strength)：
直接对当前 Token 位置所有活跃特征的 zzzz 值（ReLU 前）进行排序。用于实时诊断哪个特征正在误导模型。 
增长差值分 (Accumulated Difference)：
针对重复/死循环，计算同一个 Token 末次激活减去首次激活 的能量差。


干预方式：
\[h' \leftarrow h + \alpha d\]
d 为单位化的特征方向向量。\alpha > 0α>0\alpha > 0α>0：强化该概念（诱导行为）。
\alpha < 0α<0\alpha < 0α<0：抑制该概念（消除行为）。
其中，d是特征 ID 对应的 SAE Decoder 权重向量。
