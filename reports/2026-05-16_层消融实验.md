# 实验报告：Code-tilted A∩D 特征层消融实验

## 运行日期 & 模型信息

- **日期**: 2026-05-16
- **模型**: Qwen3-8B + SAE-Res-Qwen3-8B-Base-W64K-L0_100
- **干预方法**: add_direction α=-20
- **候选特征**: 各层独立筛选的 code-tilted A∩D（tilt≥2.0）
- **温度**: 0.3，**max_tokens**: 1024

## 背景与动机

此前所有干预实验（05-12 至 05-16）都基于一个隐含假设：**L20 是 code-tilted 特征发挥因果作用的唯一或主要层**。这一假设来自 Logits Lens 分析（L19→L20 是"using"从 18%→80% 的结构决策爆发点），以及 L26 和 L31 的初步对照实验。

但存在两个关键漏洞：

1. **L26/L31 的对照用的是 L20 筛选的特征，而不是各层自身特征**。结论"L26 无效果"可能是因为 L20 的特征在 L26 本来就不激活（跨层不通用），而非 L26 的因果作用弱。
2. **L15 从未用 add_direction 测试**。05-11 的推理重叠分析显示 L15 的拆解/概念比（1.10x）是三层中最接近 1.0 的，且 A-D 特征在 L15 的 procedural overlap 最高（19.1%），提示 L15 可能有独立或叠加的作用。

本实验的动机：拆除"仅有 L20 有效"这个未经验证的假设，对 **L15, L20, L24, L26, L31 五层做完整的层消融实验**——每层用该层自身筛选的 code-tilted A∩D 特征，统一使用 add_direction α=-20，在 5 个标准任务上测试。

## 实验方法

### 第一步：多层特征扫描

使用 `scan_shared_with_tilt.py` 一次扫描五层（L15, L20, L24, L26, L31），输入 30 条 code prompts + 15 条 description prompts，每层独立计算 tilt 值。

```bash
python3 surgenry/scan_shared_with_tilt.py \
  -l 15,20,24,26,31 --topk 200 \
  --output layer_ablation_features.json
```

### 第二步：对照实验设计

设计了**两种消融对照**以严格检验层特异性和特征特异性：

**对照 A（初版——全空间随机 FID）**：从 65536 个全部 feature ID 中均匀采样（含绝大部分 dead neuron），在 L20 用 add_direction α=-20 测试。由于 dead neuron 占绝大多数，这一对照过于宽松，意义有限。

**对照 B（严版——A-only ∪ D-only 随机采样）**：完整扫描 30 code + 15 desc prompts 在 L15 和 L20 上的全量 top-100 激活特征，构建各层完整的 A-only ∪ D-only 集合（**完全排除所有 A∩D 共享特征**）。从该集合中随机采样与 tilt≥2.0 相同数量的特征，在同等条件下测试。

对照 B 的严格性远高于对照 A：
- 采样自**真实被激活的特征空间**，而非包含大量 dead neuron 的全域均匀采样
- **排除了所有 A∩D 共享特征**，确保特征集与 code-tilted A∩D 在 A∩D 归属上严格不相交
- L15 和 L20 使用完全相同的流程，确保跨层可比

### 干预参数

- **干预方式**: add_direction
- **剂量**: α=-20（与 L20 标准剂量一致）
- **特征**: 
  - code-tilted A∩D: 各层 tilt≥2.0 的特征（L15: 21个, L20: 23个）
  - 对照 B: 各层 A-only∪D-only 中随机采样同等数量
- **温度**: 0.3
- **max_tokens**: 1024

### 任务清单

| 任务 | 提示词 | 任务类型 |
|------|--------|----------|
| Capital | "The capital of France is" | 事实检索 + 简短生成 |
| Algebra | "If 3x + 7 = 22, what is the value of 2x + 5?" | 代数推理 |

## 分析讨论

### 1. L15 的激活特征具有不可区分性

L15 最关键的发现：**无论是 code-tilted、低 tilt（tilt≈0.5x）还是完全从 A-only∪D-only 中随机采样的特征，在 α=-20 下干预全部产生无法区分的全面崩溃**。5 个任务全部退化为符号级 stutter 或 Unicode 乱码，不保留任何文本生成能力。

这一"不可区分性"意味着：**L15 的激活特征不区分 A∩D、A-only、D-only、code-tilted 或 desc-tilted——任何特征集的抑制都导致模型基础文本生成能力的全面瓦解。** L15 的特征空间作为一个整体承载语言生成的基础功能，无法通过 tilt 等指标区分出有特异因果作用的子集。

具体证据：
- **code-tilted (tilt≥2.0) vs 低 tilt (tilt≈0.5x)**：都是符号级乱码，模式不同但是都是崩溃
- **code-tilted 的 top-21 与随机 A-only∪D-only 的 21 个特征**：同样全部崩溃
- **崩溃不存在选择性**：无论抑制哪个特征子集，5 个任务同时退化

### 2. L20 的特征具有可区分的因果特异性

与 L15 相反，L20 的激活特征可以清晰地区分"有因果作用"与"无因果作用"的子集：

- **code-tilted A∩D（23 个）→ 选择性崩溃**：Algebra 保持完整推理链、Capital 概念循环但可读、Water/Clock 发生 fffff 崩溃。**不同任务产生不同模式的退化。**
- **随机 A-only∪D-only（23 个）→ 全任务一致的轻度干扰**：所有任务统一表现为 UNCLOSED 但推理内容可读，**无任务特异性**。

这一可区分性的核心含义：**L20 tilt≥2.0 的筛选明确锁定了一组与推理过程有因果关系的特征。** 相同层、相同数量、相同剂量下，非筛选特征的效应完全不同。

### 3. 筛选特征的"选择性崩溃"是因果关系的证据

L20 code-tilted 特征的特异性体现在两个层面：

**任务间选择性**：在同一剂量下，不同任务受到不同程度的影响。Algebra（短链无状态推理）完全不退化；Clock 和 Water（长链有状态推理）发生 fffff 崩溃。这种有规律的任务间差异无法用"随机噪声"解释。

**与随机对照的差异**：随机 A-only∪D-only 在全任务上产生**均匀的、一致的**轻度 UNCLOSED 效应，而 code-tilted A∩D 产生**差异化的、选择性的**崩溃模式。这种差异只能由特征的功能特异性解释。

### 4. 层次化模型

综合 L15 与 L20 的消融结果，模型的功能组织如下：

```
L15: 通用语言基础层
     特征不可区分——任何特征集抑制都全面崩溃
     特征编码基础文本生成能力
     ↓抑制效应：全面乱码（与特征选择无关）
        
L20: 结构化输出编排层（核心因果层）
     特征可区分——仅 code-tilted A∩D 有选择性效应
     特征编码"结构化输出编排"信号
     ↓抑制效应：选择性崩溃（推理脆弱、简单任务弹性）
        
L24: 过渡层
     微弱残余效应，输出为 UNCLOSED 但可读
        
L26+: 风格/内容层
     code-tilted 特征在此处已无实质因果作用
```

### 5. 对"共享泛化子空间"框架的影响

**核心结论不变**：L20 的 23 个 code-tilted A∩D 特征对推理存在特异性因果作用，这是 6 天实验的可靠发现。

**L15 数据的重新定位**：L15 的强效应在实验早期（层消融前）曾被误认为 "L20 以上也有因果作用" 的证据。clean 消融揭示了这一推论是错误的——L15 的效应是**非特异性的**（随机特征即崩），不指向推理过程。

## 结论

### 已确认的事实

1. **L15 的激活特征具有不可区分性**：无论选取 L15 的哪类特征（code-tilted A∩D、low-tilt A∩D、随机 A-only∪D-only），抑制后均导致全面不可逆的崩溃。L15 的特征作为一个整体承载基础文本生成能力。
2. **L20 的可区分性**：L20 中仅 code-tilted A∩D 特征具有选择性因果作用，同层同量的随机 A-only∪D-only 特征仅产生均匀的轻度干扰。
3. **L20 code-tilted 特征的因果特异性**：其崩溃模式是任务间有规律的（Algebra 免疫、Water/Clock 崩溃），且与随机对照的均匀效应有本质差异，证明筛选的特征与推理过程有因果关系。
4. **L24 效应微弱**（UNCLOSED 但推理可读），L26/L31 无实质因果作用。
5. **L15 的强效应不具有特异性**，不参与解释推理过程。它只是对实验方法的警示——效应强度不等于特异性和因果相关性。

### 方法学教训

| 教训 | 具体例子 |
|:----|---------|
| **跨层特征不通用** | L26 和 L31 用 L20 特征做对照不够——需要用各层自身特征 |
| **效应强度 ≠ 特异性** | L15 效应远强于 L20，但它是非特异性的；仅看强弱是不够的 |
| **消融对照的采样空间至关重要** | 从全域 65536 均匀采样（dead neuron 为主）过于宽松；必须从激活特征空间中采样，且需排除目标特征集 |
| **层消融需要 A-only∪D-only 对照** | 仅做 A∩D 内部比较不够，需要同层非共享特征证明特异性 |
